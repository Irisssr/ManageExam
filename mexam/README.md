# mexam

## 进入项目
```
cd mexam
```

## 安装包
```
npm install
或 
cnpm install
或 yarn install
（首先保证电脑已安装cnpm、yarn）
```
##### npm下载速度慢，可以使用淘宝镜像
```
npm config set registry https://registry.npm.taobao.org
```

### 项目启动
```
npm run serve
或者
yarn serve
```

### 项目打包
```
npm run build
或者
yarn build
```

### Lints and fixes files
```
yarn lint
```
